Game Demo: Meteor-eyes

This is a project that I've created to learn Unity in depth. Except for the primitive shapes and textures, everything was created by me. The character was designed, rigged and animated by me (none of these are my strenghts). All vfx were created using Unity's VFX graph and shader graph. The game uses the universal render pipeline and the built-in lighting system with some post-processing.
Some features:

  • Player respawns on death
  • Player can shoot and use a sword slash attack
  • Player can use a shield to block incoming attacks
  • Shield use has a cooldown period
  • Shields can be picked up
  • Enemies spawn in waves
  • There are two sections in the demo, each section has its own waves and enemies
  • Some enemies can shoot, and have different behaviours
  • Enemies drop health or shields according to a drop rate
  • There are different rarities for items (common, rare, super, ultra, ultimate) even though right now the only items are shields and health.
